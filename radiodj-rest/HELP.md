## RadioDJ REST API
Make sure the 'Rest Server' plugin is enabled and functioning properly.

**Available commands for RadioDJ**

* Play playlist track
* Remove playlist track
* Stop player
* Set pause state
* Restart player
* Play from intro
* Clear playlist
* Set Auto DJ state
* Set Assisted state
* Set event run state
* Refresh events
* Set input state
* Play cart by number
* Display message in RadioDJ
